# Gallery folder

Drop optimized SEP 2025 photos here.

**Recommended workflow:**
1. Compress photos with [squoosh.app](https://squoosh.app) — target 800px wide, ~80KB JPEG at quality 75
2. Name them predictably: `sep2025-01-opening.jpg`, `sep2025-02-keynote.jpg`, etc.
3. Update the `<img>` tags in `../index.html` gallery section to reference each file

See the main README for full instructions.
